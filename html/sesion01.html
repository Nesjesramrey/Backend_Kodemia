Nodejs

Es un entorno de ejecucion para Javascript construido con el motor V8 de Chrome. Escribimos codigo para qye otros humanos lo lean y lo entiendad para que sea convertido a lenguaje maquina y sea ejecutado.
Es mejor un codigo lento pero que se entienda.
Como es el compilado de codigo? Un compilador lee todo el codigo, y durante varias etapas valida la sintaxis, tipos y semantica. Si todo esta correcto crea un archivo con las instrucciones en lenguaje maquina.
Como es interpretado el codigo. El interprte lee una linea del codigo y valida la sintaxis. Si todo esta correcto convierte la instruccion a lenguaje maquina y se ejecuta. Puede haber errores de ejecucion.

Funcionamiento de NodeJs

V8-Engine (interprete)creado por Google para ejecutar Javascript y Web Assembly(las instrucciones que entiende el navegador).
Entorno de ejecucion que permite usar V8 y acceder recursos del sistema operativo.
Que no es NodeJs, no es un lenguaje, el lenguaje que se usa sigue siendo Javascript, No es un framework, No es una libreria. Es un entorno de ejecucion

Que es un servidor?

Que es NPM?

Es el administrador de paquetes por defecto de NodeJs Permite instalar los distintos paquetes de sofware en un proyecto de node. Los paquetes que requiere un proyecto se puede definir en un archivo llamado package.json
